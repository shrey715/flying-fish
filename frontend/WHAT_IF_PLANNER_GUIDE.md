# 🕹️ What-If Scenario Planner - Complete Guide

## 🎯 Overview

The **What-If Scenario Planner** is an interactive feature that transforms your dashboard from a passive reporting tool into an active decision-making tool. Retention agents can simulate interventions and instantly see their impact on churn risk.

---

## 🌟 Key Features

### 1. **Real-Time Impact Analysis**
- Adjust customer features with interactive sliders
- Model instantly recalculates churn probability
- See immediate impact of interventions

### 2. **Three Analysis Modes**

#### 🎨 Custom Adjustments
- Manual control over key features
- Test your own intervention strategies
- Real-time feedback on each change

#### 📋 Pre-defined Scenarios
- Ready-to-use retention strategies
- Industry-standard interventions
- One-click scenario testing

#### 🎯 Goal-Seeking Analysis
- Set target churn risk
- AI finds optimal intervention
- Minimum change required

## 🔧 Technical Details

### How It Works:

#### 1. **Feature Selection**
```python
# System identifies top actionable features:
- Numeric features only (can be adjusted)
- High SHAP impact (affects prediction)
- Reasonable value ranges
- Top 6-10 features selected
```

#### 2. **Real-Time Prediction**
```python
# For each adjustment:
1. Copy customer data
2. Apply changes
3. Run model.predict_proba()
4. Calculate new churn risk
5. Update UI instantly (<200ms)
```

#### 3. **Goal-Seeking Algorithm**
```python
# Binary search to find optimal value:
1. Set search range based on SHAP direction
2. Test midpoint value
3. Compare to target
4. Narrow range
5. Repeat until converged (20 iterations)
6. Return optimal value
```
